Exact Problem Statement Chosen: 
 
“Financial Literacy & Inclusion – Student Finance Management” 
This challenge focuses on enabling students to effectively 
manage their personal finances by developing digital solutions 
that promote financial literacy, track expenses, provide budget 
insights, and encourage savings habits. 
 
Solution Overview: 
 
We propose a Student Finance Management System built 
using ServiceNow, designed to help students track daily 
expenses, categorize spending, and receive budgeting insights. 
The system will feature a personalized dashboard, goal-based 
savings tracker, automated reminders, and financial tips 
powered by analytics. It aims to foster informed financial 
decisions, promoting long-term financial discipline and 
independence among students.
